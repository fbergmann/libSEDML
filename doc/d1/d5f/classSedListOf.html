◆ clear()

void SedListOf::clear ( bool  doDelete = true)

Removes all items in this SedListOf object.

If parameter doDelete is true (default), all items in this SedListOf object are deleted and cleared, and thus the caller doesn't have to delete those items. Otherwise, all items are cleared only from this SedListOf object; the caller is still responsible for deleting the actual items. (In the latter case, callers are advised to store pointers to all items elsewhere before calling this function.)

Parameters
doDeleteif true (default), all items are deleted and cleared. Otherwise, all items are just cleared and not deleted.
